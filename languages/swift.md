## Basics

Use `let` to make a constant and `var` to make a variable.

```swift
var myVariable = 42
myVariable     = 50
let myConstant = 42
```

implicit and explicit

```swift
let implicitInteger        = 70
let implicitDouble         = 70.0
let explicitDouble: Double = 70
```
